# 🎓 BTEC-Study

Welcome to my **BTEC-Study** repository! This space contains all the experiments, assignments, and projects from my studies at BTEC. It is organized by subject and module to help me track my learning progress and serve as a personal reference.

> **Note:** This repository documents my learning journey. Code quality and practices will evolve and improve across different modules over time.

## 📚 Overview
Here is what you will find in this repository:
- **Lab experiments & practice exercises**
- **Class assignments & homework**
- **Small projects & demos**
- **Notes, references, & study materials**

## 🗂️ Repository Structure
The repository is categorized by domains of study. *(Feel free to explore the individual folders for more details):*

- 💻 **`Programming/`** - Source code for programming modules (C, C++, Java, Python, etc.)
- 🤖 **`Robotics/`** - Embedded systems, Arduino Nano configurations, and motor driver (TB6612FNG/DRV8833) projects
- 🌐 **`Networking/`** - Networking labs, configurations, and reports
- 🗄️ **`Database/`** - SQL scripts, ER diagrams, and database exercises
- 🕸️ **`Web/`** - HTML/CSS/JS projects and web assignments
- 🧮 **`Math/`** - Problem sets, formulas, and notes
- 📂 **`Others/`** - General study resources and minor subjects
